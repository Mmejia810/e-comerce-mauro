e-comerce-mauro
Aplicación de e-commerce móvil y web desarrollada con Angular e Ionic. Consume una API REST para gestionar productos, categorías y usuarios. Permite explorar productos, filtrar por categorías, ver detalles y realizar compras.

Funcionalidades

Listado y exploración de productos
Filtrado por categorías
Vista de detalle de producto
Flujo de compra
Gestión de usuarios


Stack

Ionic
Angular
TypeScript
SCSS
API REST


Instalación y uso
bashgit clone https://github.com/Mmejia810/e-comerce-mauro.git
cd e-comerce-mauro
npm install
ionic serve
